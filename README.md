
## Run Locally

Clone the project

```bash
  git clone https://github.com/KyahWel/war-excel-generate.git
```

Go to the project directory

```bash
  cd war-excel-generate
```

Install dependencies

```bash
  npm install
```

Go to the client and service directory

```bash
  cd client
  cd service
```

Start the server

```bash
  npm run dev
```
